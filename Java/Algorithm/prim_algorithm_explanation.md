# Prim 알고리즘 단계별 설명 (정점 c에서 시작)

## 초기 설정
- 시작 정점: c
- 방문한 정점 집합(S): {c}
- 방문하지 않은 정점 집합(V-S): {a, b, d, e, f}
- 초기 D[v] 값:
  - D[c] = 0 (시작 정점)
  - D[a] = ∞
  - D[b] = ∞
  - D[d] = ∞
  - D[e] = ∞
  - D[f] = ∞

## 1단계: 정점 c에서 인접한 정점들의 D[v] 갱신
- c와 연결된 간선: (c,b)=1, (c,f)=1
- D[b] = min(∞, 1) = 1
- D[f] = min(∞, 1) = 1
- 현재 D[v] 값: D[a]=∞, D[b]=1, D[c]=0, D[d]=∞, D[e]=∞, D[f]=1

## 2단계: 방문하지 않은 정점 중 D[v]가 최소인 정점 선택
- D[b]와 D[f]가 모두 1로 최소값을 가짐 (동점인 경우 아무거나 선택, 여기서는 b를 선택)
- 정점 b를 S에 추가: S = {c, b}
- 방문하지 않은 정점: {a, d, e, f}

## 3단계: 정점 b에서 인접한 정점들의 D[v] 갱신
- b와 연결된 간선: (b,a)=3, (b,c)=1, (b,d)=4, (b,f)=2
- c는 이미 방문했으므로 무시
- D[a] = min(∞, 3) = 3
- D[d] = min(∞, 4) = 4
- D[f] = min(1, 2) = 1 (변화 없음)
- 현재 D[v] 값: D[a]=3, D[b]=1, D[c]=0, D[d]=4, D[e]=∞, D[f]=1

## 4단계: 방문하지 않은 정점 중 D[v]가 최소인 정점 선택
- D[f]=1이 최소이므로 f를 선택
- 정점 f를 S에 추가: S = {c, b, f}
- 방문하지 않은 정점: {a, d, e}

## 5단계: 정점 f에서 인접한 정점들의 D[v] 갱신
- f와 연결된 간선: (f,b)=2, (f,c)=1, (f,d)=7, (f,e)=9
- b와 c는 이미 방문했으므로 무시
- D[d] = min(4, 7) = 4 (변화 없음)
- D[e] = min(∞, 9) = 9
- 현재 D[v] 값: D[a]=3, D[b]=1, D[c]=0, D[d]=4, D[e]=9, D[f]=1

## 6단계: 방문하지 않은 정점 중 D[v]가 최소인 정점 선택
- D[a]=3이 최소이므로 a를 선택
- 정점 a를 S에 추가: S = {c, b, f, a}
- 방문하지 않은 정점: {d, e}

## 7단계: 정점 a에서 인접한 정점들의 D[v] 갱신
- a와 연결된 간선: (a,b)=3, (a,d)=2, (a,e)=4
- b는 이미 방문했으므로 무시
- D[d] = min(4, 2) = 2
- D[e] = min(9, 4) = 4
- 현재 D[v] 값: D[a]=3, D[b]=1, D[c]=0, D[d]=2, D[e]=4, D[f]=1

## 8단계: 방문하지 않은 정점 중 D[v]가 최소인 정점 선택
- D[d]=2가 최소이므로 d를 선택
- 정점 d를 S에 추가: S = {c, b, f, a, d}
- 방문하지 않은 정점: {e}

## 9단계: 정점 d에서 인접한 정점들의 D[v] 갱신
- d와 연결된 간선: (d,a)=2, (d,b)=4, (d,e)=5, (d,f)=7
- a, b, f는 이미 방문했으므로 무시
- D[e] = min(4, 5) = 4 (변화 없음)
- 현재 D[v] 값: D[a]=3, D[b]=1, D[c]=0, D[d]=2, D[e]=4, D[f]=1

## 10단계: 방문하지 않은 정점 중 D[v]가 최소인 정점 선택
- 남은 정점은 e 하나뿐이므로 e를 선택
- 정점 e를 S에 추가: S = {c, b, f, a, d, e}
- 방문하지 않은 정점: {}

## 최종 결과
- 모든 정점을 방문 완료
- 최종 D[v] 값: D[a]=3, D[b]=1, D[c]=0, D[d]=2, D[e]=4, D[f]=1
- 최소 신장 트리의 간선: (c,b), (c,f), (a,d), (a,e)
- 최소 신장 트리의 총 가중치: 1 + 1 + 2 + 4 = 8
